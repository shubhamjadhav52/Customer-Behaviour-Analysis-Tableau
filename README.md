Customer Analysis Dashboard (Tableau)
--Overview

This project presents an interactive Customer Analysis Dashboard built using Tableau Public. It provides insights into customer purchasing behavior, revenue distribution, and demographic trends to support data-driven decision-making.

--Features
Total Revenue Overview
Displays overall revenue: ₹716,376
Top 10 Categories
Highlights best-performing product categories. Mobiles & Tablets generate the highest revenue.
Revenue Per State
Geographic visualization of revenue distribution across India, helping identify high-performing regions.
Revenue Per Category Per Gender
Comparative analysis of male and female purchasing behavior.
Revenue Per Mode of Payment (MOP)
Breakdown of transactions by payment methods such as COD, bank transfer, and digital payments.
Month-wise Revenue Trend
Time series analysis showing revenue trends across different months.
Revenue Per Age Group
Customer segmentation by age groups, with the 30–40 group contributing the most.
Category Filter
Interactive dropdown to filter dashboard data by category.
